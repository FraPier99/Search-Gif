# 🎉 Search GIF App 🚀

Una semplice applicazione che consente di cercare GIF utilizzando l'API di [Giphy](https://giphy.com/).  
Il backend è gestito con **Netlify Functions**, che mantiene l'API Key nascosta per una maggiore sicurezza. 🔒
